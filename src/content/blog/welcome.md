This is a placeholder post. Replace the content in `src/content/blog/` with your new
markdown files - one file per post. Each post needs the frontmatter fields shown at the
top of this file: `title`, `description`, `date`, `category`, `tags`, and `draft`.

The image above shows up as this post preview thumbnail on the blog index page - the
first image found in a post body is used automatically so that no separate cover-image field is needed. Posts with no image just show the title, date, and description instead.

`category` is the single label used for the filter pills on the blog index. `tags` is a separate, more granular list shown on each post - use it for anything more specific than the category.
